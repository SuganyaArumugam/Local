How to use:

1. Open web browser. Go to https://www.gmail.com/ in your computer's web browser. This will open your Gmail inbox if you're logged in. •If you aren't logged in, you'll need to enter your email address and password when prompted.

2. Make sure you're using the most current Gmail inbox. Do the following: i) Click the "Settings" gear Image titled Android7settings.png . ii) Click Try the new Gmail at the top of the drop-down menu. •If you see Go back to classic Gmail in the drop-down menu, you're already using the new version of Gmail.
